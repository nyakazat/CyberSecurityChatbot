# CyberSecurityChatbot

The Cybersecurity Awareness Chatbot – NYAKAZA is a C# console-based application developed to educate South African citizens about common cyber threats. The chatbot simulates a conversational assistant that provides practical guidance on how to stay safe online, focusing on topics such as phishing, password security, and safe browsing.
It demonstrates foundational programming concepts including user input handling, string manipulation, automatic properties, multimedia integration, and version control using GitHub with Continuous Integration (CI)
